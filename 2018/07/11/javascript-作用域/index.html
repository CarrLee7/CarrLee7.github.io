javascript 作用域

CarrLee7
2018-07-11
javascript

引言

作用域是javascript的基础之一，如果你还不了解作用域及相关知识就继续往下看吧

作用域

声明变量的位置决定了变量的作用域

这个例子中，变量a在函数test()里声明，所以变量a的作用域为整个函数体

在两个函数体中，同名变量互不影响

内部函数可以访问到外部函数的变量，反之则不行

当内部变量和外部变量重复时，内部变量会覆盖外部变量

全局作用域

不在函数内定义的变量具有全局作用域

顶层函数的定义也是全局变量

javascript只有一个全局作用域，如果变量没有在当前作用域找到，会继续向上查找，如果全局作用域也没有找到，会抛出错误 ReferenceError

for语句中无法定义具有局部作用域的变量

变量提升

javascript函数会扫描所有的变量声明，将所有的声明“提升”到函数顶部，但赋值不会提升
